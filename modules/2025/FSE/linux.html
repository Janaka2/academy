A. Foundations of Linux

Filesystem Hierarchy

  • /etc – config files
  • /var/log – logs
  • /opt – optional software (great for JARs)

Paths

cd /opt/apps     # absolute path
cd ../logs       # relative path

Navigation Commands

ls -la
pwd
tree -L 2

B. File & Directory Operations

Copy, Move, Remove

cp -rv target/*.jar /opt/apps/
mv app.log logs/
rm -rf temp/

Compression

tar -czvf backup.tar.gz /opt/apps/

Compare

diff application.yml application-prod.yml
vimdiff settings.xml settings-dev.xml

C. Search & Filtering

Find Stacktraces, Configs, Keys

grep -i exception logs/app.log
find /var -name "*.yml"
locate application.properties
xargs -a services.txt grep -H 'timeout'

Pipelines

cat log.txt | grep ERROR | cut -d' ' -f5- | sort | uniq -c

D. Text Processing

Practical Usage

cut -d":" -f1 /etc/passwd
awk '{print $2, $5}' performance.csv
sed 's/localhost/127.0.0.1/g' config.yml

Tail Logs Live

tail -f /var/log/spring.log | grep WARN

E. Working With Processes

List and Kill

ps aux | grep java
kill -9 <PID>
lsof -i :8080

F. Networking Tools

API Testing

curl -X POST http://localhost:8080/api -d '{"test":true}'
wget https://repo.maven.org/artifact.jar

Ports & DNS

netstat -tulnp
ping google.com
traceroute github.com

rsync -avz /var/log/ user@host:/data/logs

N. 20 Real Java Scenarios

  • Find what’s using port 8080 → lsof -i :8080
  • Tail Spring logs live → tail -f spring.log
  • Search huge log without loading → grep Exception big.log
  • Copy logs between servers → rsync -avz /logs user@host:/backup
  • Kill hung Spring app → pkill -f spring
  • Check memory usage → free -h
  • View running containers → docker ps
  • View CPU usage → top
